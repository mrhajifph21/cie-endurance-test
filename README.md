## 🧵 The Concept
This website is built not just as a standard portfolio, but as a literal representation of my identity as a Full Stack Developer and a Creative. Taking heavy inspiration from Cretivox's #CIERAW campaign, the UI translates physical streetwear elements—raw denim textures, price tags, and stitches—into a digital, interactive experience.

## 🎯 Objectives Completed

### 1. The Stack
Built robustly using **Next.js** and **Tailwind CSS**, fully responsive across devices. 

### 2. The GSAP Experience
Heavy emphasis on smooth, interactive animations to make the page feel alive:
- **The Swing:** Mouse-tracking physics on the initial Price Tag gate.
- **The Snip:** Fall-away animation upon successful API authentication.
- **The Ripped Identity:** `ScrollTrigger` tear effect revealing the About section.
- **The Stitches:** SVG path-drawing animation acting as section dividers.
- **The Loop:** Seamless, hardware-accelerated marquee gallery.

### 3. The Fierce Interaction
Integrated the mandatory 3 "Fierce" portraits (Front, Left, Right) seamlessly using mouse-coordinate detection, allowing the portrait to track the user's cursor dynamically.
